打包构建工具，一个脚手架应用必不可缺的一部分。

这里我们选择常用的webpack，可以通过它实现以下诸多功能

- react jsx/tsx编译、babel编译、sass样式表编译
- 代码压缩、tree shaking、图片压缩、svg转组件
- 开发热更新、性能优化
- source-map 源码映射
- 路径映射
  以及更多的功能
